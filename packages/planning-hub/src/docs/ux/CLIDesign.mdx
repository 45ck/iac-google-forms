# CLI Design

## Design Principles

| Principle                  | Implementation                                  |
| -------------------------- | ----------------------------------------------- |
| **Clarity**                | Show diffs before changes, require confirmation |
| **Progressive Disclosure** | Essential info default, `--verbose` for details |
| **Visual Language**        | `✓` success, `✗` error, `+/-/~` diff symbols    |
| **CI-Friendly**            | Exit codes, `--auto-approve`, markdown output   |
| **Helpful Errors**         | What, Why, How to fix                           |

## Color Usage

| Color  | Usage                   |
| ------ | ----------------------- |
| Green  | Success, additions      |
| Red    | Errors, removals        |
| Yellow | Warnings, modifications |
| Cyan   | URLs, hints             |
| Gray   | Secondary info          |
